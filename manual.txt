Manual Pessoal - Lógica do Modal

1. O que é o Modal?
   - Janela que aparece sobre o conteúdo da página para exibir informações adicionais.

2. Estrutura do Modal:
   - HTML: Estrutura (divs, título, descrição, links, botão de fechar).
   - CSS: Estilo (posicionamento, cores, animações).
   - JavaScript: Lógica para abrir, fechar e preencher o modal.

3. Passo a Passo da Lógica:

   Passo 1: Selecionar os Elementos do Modal
   - Seleciona a div do modal, título, descrição, links e botão de fechar.

   Passo 2: Selecionar os Links dos Projetos
   - Seleciona todos os links que abrem o modal e os combina em um array.

   Passo 3: Adicionar Eventos aos Links
   - Adiciona um evento de clique a cada link para abrir o modal.

   Passo 4: Buscar o Projeto no Banco de Dados
   - Usa o data-id do link para encontrar o projeto correspondente no array dataBaseProjetos.

   Passo 5: Preencher o Modal com as Informações
   - Define o título, descrição e links do modal com os dados do projeto.
   - Exibe o modal alterando o estilo display para flex.

   Passo 6: Fechar o Modal
   - Fecha o modal ao clicar no botão de fechar ou fora do conteúdo.

4. Resumo:
   - Abrir: Clica no link → Busca projeto → Exibe modal.
   - Fechar: Clica no botão de fechar ou fora do modal.

  function mostrarNovoConteudo(item) {
    const target = item.getAttribute("data-item");

    // Oculta todos os conteúdos e exibe apenas o correspondente ao item
    conteudoItens.forEach((conteudo) => {
      if (conteudo.getAttribute("data-conteudo") === target) {
        conteudo.classList.add("active");
      } else {
        conteudo.classList.remove("active");
      }
    });

    // Libera o bloqueio após a transição
    emTransicao = false;
  }

  const homeItem = document.querySelector('.menu-item[data-item="home"]');
  const homeConteudo = document.querySelector('.conteudo-item[data-conteudo="home"]');

  // Ativa o "Home" ao carregar a página
  if (homeItem && homeConteudo) {
    homeItem.classList.add("active");
    homeConteudo.classList.add("active");
  }

  menuItens.forEach((item) => {
    item.addEventListener("click", () => {
      if (emTransicao) return; // Bloqueia cliques durante a transição

      emTransicao = true; // Inicia o bloqueio

      // Remove a classe "active" de todos os itens do menu
      menuItens.forEach((menuItem) => menuItem.classList.remove("active"));
      item.classList.add("active");

      // Remove o conteúdo ativo atual (se houver)
      const activeConteudo = document.querySelector(".conteudo-item.active");
      if (activeConteudo) {
        activeConteudo.classList.remove("active");
      }

      // Agenda a exibição do novo conteúdo após o tempo de transição
      setTimeout(() => {
        mostrarNovoConteudo(item);
      }, tempoTransicao);
    });
  });
}
