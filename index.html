O QUE EU SEI FAZER?

Back-End

    Linguagens:
  • Node.js
    Frameworks:
  • Express.js para criação de APIs RESTful.
  • Conhecimento de GraphQL
  • Uso de ES6+ (arrow functions, destructuring, async/await, etc.).
    Banco de Dados:
  • Bancos relacionais: MySQL, PostgreSQL.
  • Bancos não relacionais: MongoDB.
    APIs:
  • Criação e consumo de APIs REST.
  • Autenticação e autorização (JWT, OAuth).
    Ferramentas:
  • Uso de Docker para conteinerização.
  • Conhecimento básico de CI/CD (GitHub Actions, GitLab CI, etc.).
    Segurança:
  • Boas práticas de segurança (CORS, sanitização de inputs, etc.).

Front-End

    HTML/CSS:
  • Dominio de HTML semantico.
  • Estilização com CSS puro, incluindo Flexbox e Grid.
  • Uso de Pré-processadores como SASS/SCSS.
    JavaScript:
  • Conhecimento sólido em JavaScript Vanilla.
  • Manipulação do DOM.
  • Uso de ES6+ (arrow functions, destructuring, async/await, etc.).
    Frameworks e Bibliotecas:
  • React.js (componentes, hooks, context API, React Router).
  • Uso de bibliotecas como Axios para requisições HTTP.
  • Vue.js
    Ferramentas de Build:
  • Webpack, Vite ou similares.
  • Uso de npm/yarn para gerenciamento de pacotes.
    Responsividade:
  • Design responsivo com media queries.
  • Uso de frameworks como Bootstrap ou Tailwind CSS.
    Testes:
  • Testes end-to-end com Cypress

UX/UI

    Design Thinking:
  • Abordagem centrada no usuário.
  • Pesquisa de usuários e personas.
    Wireframes e Protótipos:
  • Uso de ferramentas como Figma, Adobe XD ou Sketch.
  • Criação de wireframes de baixa e alta fidelidade.
    Usabilidade:
  • Testes de usabilidade (A/B testing, heatmaps).
  • Princípios de design como hierarquia visual, contraste e alinhamento.
    Acessibilidade:
  • Conformidade com WCAG (Web Content Accessibility Guidelines).
  • Uso de ARIA roles e atributos para melhorar a acessibilidade.
  • Fluxo de Usuário:
  • Criação de jornadas do usuário (user journeys).
  • Análise de pontos de atrito e melhoria de fluxos.
    Ferramentas:
  • Uso de ferramentas de análise como Google Analytics ou Hotjar.
  • Prototipagem interativa para validação de ideias.
