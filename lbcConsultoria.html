Introdução

A LBC CONSULTORIA é uma empresa especializada em planos de saúde e seguros com mais de 15 anos de mercado. Seu principal objetivo é ajudar famílias e empresas a encontrar a proteção ideal, com soluções personalizadas e suporte contínuo.

Objetivo

O projeto foi realizado para um site moderno e responsivo para a LBC Consultoria, com foco na captação de leads. A melhoria da experiência do usuário era o objetivo principal, alinhando-o à identidade visual da empresa, garantindo uma navegação intuitiva e uma alta taxa de conversão.

Estratégia

O projeto foi dividido em duas fases principais: UX/UI e Desenvolvimento. Na etapa de UX/UI, focou na pesquisa de mercado, persona, jornada do usuário, wireframes e protótipos. Na fase de desenvolvimento, trabalhou na implementação do front-end e back-end, além da integração com a API do Google Sheets.

Metodologia

Durante o projeto, realizou pesquisas conduzidas por consultores de UX/UI, com a ajuda de ferramentas de posicionamento estratégico de botões e priorização de gestão (para organização visual das informações). Esses conceitos ajudaram a criar uma experiência de usuário simplificada e eficiente.

Tecnologias Utilizadas

  • Vite: Para criar um ambiente de desenvolvimento rápido e eficiente.
  • React: Para estruturar a interface do usuário com componentes reutilizáveis.
  • GIT: Para controle de versão e colaboração.
  • HTML/CSS: Para estruturação e estilização do site.
  • JavaScript: Para lógica e interatividade da aplicação.

Desenvolvimento de Componentes

Desenvolveu componentes reutilizáveis em React, como botões, cards de serviços e menus de navegação. Isso permitiu uma manutenção mais fácil e um código mais organizado.

Integração com Back-End

O front-end foi integrado ao back-end através da API do Google Sheets. Utilizou requisições HTTP para enviar os dados dos formulários, garantindo que as informações preenchidas pelos usuários fossem armazenadas corretamente.

Desafios e Soluções

Um dos principais desafios foi garantir que o site carregasse rapidamente, mesmo com imagens e vídeos grandes. Para isso, utilizou técnicas de otimização de desempenho, como lazy loading e code splitting, reduzindo o tempo de carregamento do site em 30%.

Resultados

Melhorou a experiência do usuário, com base nos dados coletados, foram implementadas páginas com alta taxa de saída, permitindo ajustes no design e no conteúdo. Otimizou o desempenho do site usando lazy loading e code splitting, reduzindo o tempo de carregamento do site em 30%. Tomada de decisão baseada em dados: Os analistas puderam tomar decisões mais informadas sobre o design e a funcionalidade do site.
